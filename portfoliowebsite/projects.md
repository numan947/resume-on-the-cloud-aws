### <span style="color:#7f7f7f;font-weight:normal;"><u>2. Single Shot MultiBox Detector in PyTorch</u></span>
<div style="margin-bottom:10px"> 
<img src="https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54" height="25"> 
<img src="https://img.shields.io/badge/PyTorch-%23EE4C2C.svg?style=for-the-badge&logo=PyTorch&logoColor=white" height="25"> 
</div>
##### <span style="color:#264653;font-weight:normal;">Description:</span>
<ul style="font-size:17px">
  <li>
    Implemented a replication of Single Shot MultiBox Detector (SSD) in PyTorch.
  </li>
  <li>
    Structured the project into six main parts:
    <ul>
      <li>Reading, preprocessing, and loading the data.</li>
      <li>Model design and implementation, including loss formulation.</li>
      <li>Training the model with validation and hyperparameter tuning.</li>
      <li>Processing model outputs using Non-Maximum Suppression (NMS).</li>
      <li>Evaluating the model using mean Average Precision (mAP) on testing data.</li>
      <li>Implementing object detection in images, videos, and live feeds.</li>
    </ul>
  </li>
  <li>
    Utilized PascalVOC 2007 and 2012 datasets for data parsing and processing.
  </li>
  <li>
    Built the model architecture following a referenced tutorial and structured the project accordingly.
  </li>
  <li>
    Trained the model with 300x300 and 500x500 images, achieving promising results.
  </li>
  <li>
    Learned and implemented various techniques during the project, including:
    <ul>
      <li>Atrous convolution for enlarging the field of view of filters.</li>
      <li>Usage of NVIDIA-Apex for half-precision training to speed up training using tensor cores.</li>
    </ul>
  </li>
  <li>
    Developed modular code for better management and debugging during the training process.
  </li>
